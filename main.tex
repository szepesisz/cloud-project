\section{EREDMÉNY}

Egy tömör, de látványos bemutató a projekt eredményéből. Minimális változtatást végzek a .drone.yml fileon. A push előtt és után lekérem az aktuális alkalmazásverziót. 

\begin{lstlisting}
szepesisz@neo:~/Documents/OE/lora/LoRaWAN-sniffer$ curl --silent http://localhost:8103/openapi.json | jq -r .info.version
0.2.1.dev68

szepesisz@neo:~/Documents/OE/lora/LoRaWAN-sniffer$ git add .drone.yml && git commit -m "update drone.yml" && git push gitea develop
...
szepesisz@neo:~/Documents/OE/lora/LoRaWAN-sniffer$ date
Wed Apr  8 10:28:11 PM CEST 2026

\end{lstlisting}

A pipeline lefutása után:

\begin{lstlisting}

szepesisz@neo:~/Documents/OE/lora/LoRaWAN-sniffer$ curl --silent http://localhost:8103/openapi.json | jq -r .info.version
0.2.1.dev69
szepesisz@neo:~/Documents/OE/lora/LoRaWAN-sniffer$ date
Wed Apr  8 10:29:31 PM CEST 2026

\end{lstlisting}

A folyamat tapasztalatom szerint stabilan működik, a további munkám során a kutatócsoportban, valamint a diplomamunkában jelentős segítség lesz, valamint a projektvezetők is pozitívan fogadták a környezet fejlesztését.
